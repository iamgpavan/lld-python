Code examples for the **LLD Chronicles** — a Medium series on Low-Level Design in Python.

|-----------|--------|---------|

| D — Dependency Inversion | solid/dip/ | Coming soon |

## How to Use

Each folder contains:
- `bad_example.py` — code that violates the principle
- `good_example.py` — the refactored, compliant version
- `README.md` — links to the full article

```bash
python solid/srp/bad_example.py
python solid/srp/good_example.py
```
